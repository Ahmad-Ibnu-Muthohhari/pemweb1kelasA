Contoh ini berisi materi dari Tag Div & Span, Pengaturan Gambar, Description List, dan Internal Link.

1. Tag Div & Span

Tag <div> digunakan sebagai container untuk memisahkan struktur.

Tag <span> digunakan untuk menandai bagian teks tertentu dalam satu baris.

3. Description List

HTML
HyperText Markup Language, bahasa untuk membuat struktur halaman web.
CSS
Cascading Style Sheets, digunakan untuk mengatur tampilan halaman web.
JavaScript
Bahasa pemrograman yang digunakan untuk membuat halaman web interaktif.
